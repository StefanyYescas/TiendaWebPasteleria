#  Tienda Web de Pastelería

Proyecto de la materia **Taller de Base de Datos**.  
Esta aplicación web permite a los usuarios explorar productos de pastelería, agregarlos al carrito y realizar pedidos.  
El proyecto integra un **frontend interactivo**, un **backend funcional** y **scripts SQL** para ver los scripts que se hicieron para la base de datos, mostrando la conexión completa entre la interfaz, la lógica del servidor y el almacenamiento de datos.

---

## Tecnologías utilizadas

- **Frontend:** Vue.js / HTML / CSS / JavaScript  
- **Backend:** Node.js  
- **Base de datos:** MySQL / Scripts SQL  
- **Control de versiones:** Git / GitHub
